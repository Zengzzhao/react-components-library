# Updater 版本更新

智能版本更新检测系统，通过自动化版本对比机制，实现Web应用的无感知热更新。无需用户手动刷新页面，自动完成版本同步，保障用户始终使用最新版本的应用。

## Pre-Installation

在安装 `@hkx-plugin` 之前，请先确保项目中存在以下依赖：

```bash
pnpm add react
pnpm add antd
pnpm add @ant-design/icons
```

## Installation

```bash
pnpm add @hkx-plugin
```
