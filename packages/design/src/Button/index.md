## API

| 属性     | 说明     | 类型                                      | 默认值    |
| -------- | -------- | ----------------------------------------- | --------- |
| type     | 按钮类型 | `primary` \| `dashed` \| `text` \| `link` | `primary` |
| size     | 按钮大小 | `small` \| `middle` \| `large`            | `middle`  |
| disabled | 是否禁用 | `boolean`                                 | `false`   |
| onClick  | 点击回调 | `() => void`                              | -         |
